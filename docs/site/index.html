Fetch structured, AI-ready content from YouTube videos.

TubeFetch is a Python tool that extracts structured, AI-ready content from YouTube videos. Given one or more video IDs, URLs, playlists, or channels, it produces normalized metadata, transcripts, and optional media in formats optimized for downstream AI/LLM pipelines (summarization, fact-checking, RAG, search indexing, etc.). It provides content hashes for change detection, optional token count estimates, and unified video bundles. The tool supports both CLI and library usage with batch processing, intelligent caching, configurable retries via gentlify, and rate limiting.

Quick Start

Installation

pip install tubefetch

Fetch Metadata & Transcript

tubefetch dQw4w9WgXcQ

Fetches metadata, transcript, and exports to JSON and plain text formats.

Download Video

tubefetch dQw4w9WgXcQ --download video

Downloads the video (merged video+audio MP4) along with metadata and transcript.

Features

AI-Ready Features

LLM-Ready Text

Intelligent paragraph chunking with configurable gap thresholds, optional timestamps, and optimized formatting for AI/LLM consumption.

Content Hashing & Token Counting

SHA-256 hashes for change detection on metadata and transcripts, plus optional token count estimation via tiktoken for LLM cost planning.

Playlist & Channel Resolution

Resolve entire playlists and channels to video IDs with configurable max_videos limiting for batch processing.

Unified Video Bundles

Single JSON file per video combining metadata, transcript, errors, content hash, and token count for streamlined AI pipelines.

Core Features

Structured Metadata

Extract title, channel, duration, tags, and upload date via yt-dlp or YouTube Data API v3.

Multi-Format Transcripts

Fetch transcripts with language preference and fallback, export as JSON, plain text, WebVTT, or SubRip.

Batch Processing

Process multiple videos concurrently with per-video error isolation and progress tracking.

Operational Features

Intelligent Caching

Skip already-fetched data automatically; use --force for selective re-fetching.

Resilient Retry Logic

Powered by gentlify with exponential backoff, jitter, and smart classification of transient vs. permanent errors.

Rate Limiting

Token bucket algorithm shared across workers prevents API throttling and service errors.

Dual Interface

Use as a CLI tool for quick extraction or import as a Python library for programmatic workflows.
